## 🎨 Customization

### 🎭 Change Colors
Edit CSS variables in `style.css`:

```css
:root {
    --primary: #667eea;      /* Main color */
    --secondary: #764ba2;    /* Secondary color */
    --accent: #f5576c;       /* Accent color */
    --dark: #1a1a2e;         /* Dark color */
    --light: #f7f7f7;        /* Light color */
}
```

**Popular Color Schemes:**
- 💙 Blue & Purple: `#667eea` + `#764ba2`
- 🔴 Red & Pink: `#f5576c` + `#f093fb`
- 🟢 Green & Teal: `#00f2fe` + `#4facfe`
- 🟠 Orange & Yellow: `#ff9a56` + `#ff6a88`

### ✏️ Edit Content
1. Open `index.html` in a text editor
2. Find the section you want to edit
3. Update text, links, and images
4. Save and refresh browser

### 🔧 Add More Services
```html
<div class="service-card">
```
