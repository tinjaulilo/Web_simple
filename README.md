# Digital Studio - Creative Solutions

A modern, responsive landing page built with HTML, CSS, and JavaScript. Perfect for showcasing creative services and portfolio projects.

## Features

✨ **Modern Design**
- Beautiful gradient backgrounds
- Smooth animations and transitions
- Professional color scheme
- Clean and organized layout

📱 **Fully Responsive**
- Mobile-first approach
- Works perfectly on all devices
- Hamburger menu for mobile navigation
- Optimized for touch interactions

⚡ **Interactive Elements**
- Smooth scrolling navigation
- Hover effects on cards
- Intersection observer for scroll animations
- Form submission handling

🎯 **Key Sections**
- **Hero**: Eye-catching landing section
- **Services**: Showcase your services
- **Portfolio**: Display your projects
- **Contact**: Get in touch form
- **Footer**: Social links and info

## File Structure

```
Web_simple/

└── README.md       # Documentation
```

## Getting Started

1. Clone the repository
2. Open `index.html` in your browser
3. Customize the content, colors, and text to match your needs

## Customization

### Colors
Edit the CSS variables in `style.css`:
```css
:root {
    --primary: #667eea;
    --secondary: #764ba2;
    --accent: #f5576c;
    /* ... more colors */
}
```

### Content
Edit the text and sections in `index.html` to match your needs.

### Services
Add or remove service cards in the Services section.

### Portfolio
Update portfolio items with your own projects.
